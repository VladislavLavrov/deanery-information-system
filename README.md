# Информационная система «Деканат университета»

## Описание проекта

Данный репозиторий содержит учебный сквозной проект разработки информационной системы **«Деканат университета»**, выполняемый в рамках дисциплины **«Основы алгоритмизации и программирования»**.

Проект демонстрирует полный жизненный цикл разработки программного обеспечения информационных систем:

* анализ предметной области;
* проектирование архитектуры системы;
* моделирование базы данных;
* разработку программного кода;
* создание пользовательского интерфейса;
* аналитическую обработку данных;
* формирование отчетности;
* тестирование и контроль качества;
* логирование и обработку ошибок;
* подготовку программного продукта к развертыванию.

Результатом проекта является прототип информационной системы, предназначенной для автоматизации деятельности деканата университета.

---

# Предметная область

Информационная система предназначена для поддержки работы деканата и обеспечивает хранение и обработку информации о:

* студентах
* учебных группах
* дисциплинах
* преподавателях
* результатах обучения
* успеваемости студентов
* аналитических показателях обучения

Система позволяет выполнять следующие основные операции:

* ведение справочников студентов и групп
* управление учебными дисциплинами
* учет результатов экзаменов и зачетов
* расчет показателей успеваемости
* формирование аналитических отчетов
* визуализацию статистики обучения

# Архитектура системы

Программная система реализована на основе **многослойной архитектуры**.

Основные слои системы:

* **Domain Layer** — модели предметной области
* **Application Layer** — прикладная логика системы
* **Infrastructure Layer** — работа с базой данных и внешними сервисами
* **Presentation Layer** — пользовательский интерфейс

Такой подход обеспечивает:

* разделение ответственности компонентов
* упрощение тестирования
* масштабируемость системы
* удобство сопровождения программного кода
